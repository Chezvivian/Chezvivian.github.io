今天有机会和翻译系的老师们分享一下去年教改项目的一些最新成果，也是我近一年以来在外语教学方面使用 AI 工具的一些探索和思考。放上来希望对更多的外语教师们有所帮助。

最近的灵感源泉要感谢陈世荣老师，去听了世荣老师的一节商英课程，学到了今天分享里提到的两种很关键的呈现方式：单词导学的音频，以及对话气泡的视频博客。虽然我在以往教学中也用了类似的形式，但是陈老师的呈现效果非常惊艳，而且和我使用的工具不同，又有了新的体验。之后，结合我最近用 html 做课件的经验，又有了用轻量级 html 来改革这些音视频创建流程的想法。

今天的分享围绕三个方向展开：

**一、单词导学音频 HTML** 把单词列表交给 AI，直接生成可翻页、可朗读的 HTML 网页——每个单词一页，带中英文例句，读三遍后自动跳转下一页。借助 Microsoft Edge 内置的丰富英文音色和 Chrome 的录屏功能，几分钟内就能产出一份自然流畅的单词导学音频。这套提示词流程已整合为可重复调用的模板，方便日常批量生产。

**二、课文知识视频播客** 介绍了三种将课文转化为音视频材料的路径：一是 Google NotebookLM，上传文件后一键生成有情绪色彩的知识背景播客，免费且语音自然，适合快速备课；二是 MoviePy 本地生成方案，结合微软免费 TTS 合成声音、Pillow 绘制气泡对话、MoviePy 合成视频，内容可定制，全程本地运行；三是 HTML 对话播客，直接调用浏览器语音，文件体积极小，适合即时展示。

**三、交互式 AI 智能体数字人** 探索了 HeyGen Live Avatar 和 D-ID 两款数字人平台作为口语练习伴侣的可行性。前期调研显示学生接受程度较高，但问卷数据表明，一个月内的短期训练对降低口语焦虑尚未体现出显著效果，该方向仍有较大探索空间。

外语课程的核心需求是优质的音视频学习材料，以及能激发学生主动开口的练习场景。这次分享想说明的是：AI 并不只是一个写文案的工具——从课文到脚本、从脚本到音视频，这一套工作流现在已经可以在日常教学中形成一个完整的闭环了。
